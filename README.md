# üêö Personal dotfiles

Configuration files for my development environment.

Even though my preferences might not perfectly match what you need, you can use
the structure of this repository as a guide to creating your own dotfiles.

## Installation

Run the initialization script to symlink configurations to their appropriate
locations:

```sh
init.sh [zsh] [bash]
```

## Git Configuration

- Aliases and configuration for [Git Town](https://www.git-town.com)
- Sign commits and tags default
- Consistent line endings

## Shell Configuration

- [Powerlevel10k](https://github.com/romkatv/powerlevel10k)-inspired terminal
  prompt
- Works with both Zsh and Bash
- Supports any color theme through terminal settings
- Styling is confined to `.zshrc` and `.bashrc`
